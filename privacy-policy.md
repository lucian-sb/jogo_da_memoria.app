# Politica de Privacidade

Este aplicativo nao coleta, armazena ou compartilha dados pessoais dos usuarios.

Nao sao coletadas informacoes como nome, e-mail, localizacao ou dados do dispositivo.

O aplicativo funciona de forma totalmente offline.
