# Arquivos de Som Necessários

Para que os efeitos sonoros funcionem, adicione os seguintes arquivos nesta pasta:

- `flip.mp3` - Som ao virar uma carta
- `match.mp3` - Som ao acertar um par
- `win.mp3` - Som ao completar um nível

**Nota:** Se os arquivos não existirem, o jogo continuará funcionando normalmente, apenas sem os efeitos sonoros.
